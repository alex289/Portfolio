This web app is not just a static page but a full-functional blogging platform.
Deeply inspired by [leerob](https://github.com/leerob/leerob.io) ‚ù§.

### Overview üìù

The core packages and frameworks used in this project.

- Framework: [Next.js](https://nextjs.org/)
- Database: [PlanetScale](https://planetscale.com)
- ORM: [Prisma](https://prisma.io/)
- Authentication: [NextAuth.js](https://next-auth.js.org/)
- Deployment: [Vercel](https://vercel.com)
- Content: [Sanity](https://sanity.io)
- Styling: [Tailwind CSS](https://tailwindcss.com/)

### Testing üß™

To make sure everything is working I test my application with [Cypress](https://www.cypress.io/) and [GitHub Workflows](https://docs.github.com/en/actions/using-workflows).
Additionally [Checkly](https://www.checklyhq.com/) monitores everything at certain intervals.

To get a better developer experience I use [ESLint](https://eslint.org/) and [Prettier](https://prettier.io/).

### Features ‚ú®

- Localization (German and English translation)
- Responsive design
- SEO optimized
- Spotify currently listening integration
- monitoring page (admin only)

### How to clone this site? üì•

Make sure to have Git, NodeJs (v16) and yarn installed.
Create a `.env` file similar to [.env.example](https://github.com/Alex289/Portfolio/blob/main/.env.example)

```bash
$ git clone https://github.com/Alex289/Portfolio
$ cd Portfolio
$ yarn

To start developing run
$ yarn dev
```

Please remove my personal information before deploying your version.
