## Essentielles

> Bevor du anfängst gehe sicher, dass du die neuste Version und Updates hast.

- Schalte die Firewall in deinen Einstellungen ein! Sehr wichtig
- Installiere XCode

```bash
xcode-select --install
```

- Installiere den Paketmanager [Homebrew](https://brew.sh)

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

- Installiere die essentiellen Umgebungen und CLIs die du brauchst

```bash
brew install git node neovim
```

- Ich empfehle sehr die git commits zu signieren. Lies [hier](https://withblue.ink/2020/05/17/how-and-why-to-sign-git-commits.html) wie und wieso du das tun solltest.
- Jetzt ist es Zeit deinen Projektsammelordner `~/code` or `~/dev` zu erstellen und deine Projektrepositoies lokal zu klonen.

## Apps & Programme

Installiere alle deine Apps und Programme einfach mit brew. Das ist eine Liste von allen Dingen die ich installiert habe:

```bash
brew install --cask iterm2 visual-studio-code google-chrome notion spotify rectangle alt-tab devtoys postman stats 1password
```

- [iTerm2](https://iterm2.com) - Terminal Emulator
- [Visual Studio Code](https://code.visualstudio.com) - Code Editor
- [Google Chrome](https://www.google.com/chrome) - Browser zum entwickeln
- [Notion](https://www.notion.so/de-de) - Tool für Notizen, Dokumentationen und mehr
- [Spotify](https://spotify.com) - Music Anbieter
- [Rectangle](https://rectangleapp.com) - Window-Management-Tool für macOS
- [Alt-Tab](https://alt-tab-macos.netlify.app) - Windows alt-tab für macOS
- [Devtoys](https://github.com/ObuchiYuki/DevToysMac) - Nützliche Entwickler Tools
- [Postman](https://www.postman.com) - Plattform um APIs zu bauen und zu testen

- [Raycast](https://www.raycast.com) - Besseres Spotlight
- [1Password](https://1password.com) - Password Management

## Terminal

> Folgendes erfordert Zsh was bei MacBooks seit 2019 Standard ist.

- Installiere [Oh My Zsh](https://ohmyz.sh)

```bash
sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"
```

- Installiere das [Powerlevel10k](https://github.com/romkatv/powerlevel10k) Theme

```bash
git clone --depth=1 https://github.com/romkatv/powerlevel10k.git ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k
```

- Installiere und setze eine [Nerd Font](https://www.nerdfonts.com) deiner Wahl
- Starte deinen Terminal neu un konfiguriere Powerlevel10k

```bash
p10k configure
```

- Updatede deine `~/.zshrc` Konfiguration nach belieben. Ein Beispiel wie ich alles konfiguriert habe ist in meinem [dotfiles repo](https://github.com/alex289/dotfiles/tree/main/mac).

## System Konfigurationen

- Alle Datei Typen standardmäßig anzeigen

```bash
defaults write NSGlobalDomain AppleShowAllExtensions -bool true
```

- All Datei Typen anzeigen

```bash
defaults write -g AppleShowAllExtensions -bool true
```

- Status und Pfad anzeigen

```bash
defaults write com.apple.finder ShowPathbar -bool true
defaults write com.apple.finder ShowStatusBar -bool true
```

- Keine DS_Store Dateien erstellen

```bash
defaults write com.apple.desktopservices DSDontWriteNetworkStores -bool true
defaults write com.apple.desktopservices DSDontWriteUSBStores -bool true
```

- Alle versteckten Dateien anzeigen

```bash
defaults write com.apple.Finder AppleShowAllFiles true
```
